# TechnoHacks Certificate Verification Portal

A simple web-based certificate verification system for TechnoHacks Solutions Pvt. Ltd.

## Features
- Certificate ID validation
- Dynamic certificate details display
- Responsive design
- User-friendly interface

## How to Use
1. Open `technohacks.html` in a web browser
2. Enter a valid certificate ID
3. Click "Verify" to view certificate details

## Valid Certificate IDs
- TH1064662
- TH1065731
- TH1044398
